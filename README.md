## General info
It's a TimeTable Management System for ESSAT University of Gabes 
	
## Technologies
Project is created with:
* Java version: 1.8
* Bootstrap version: 3
* Genetic Algorithm
	
## Setup
To run this project, install it locally using maven:

```
$ cd ../ESSAT-TimeTableManagementSystem
$ mvn install
```
Run in Browser
